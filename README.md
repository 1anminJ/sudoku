## 기능

|  | 설명 |
|--|------|
| 🎯 **난이도 3단계** | 쉬움 / 보통 / 어려움 |
| ⏱ **타이머** | 일시정지 / 재개 지원 |
| 💡 **힌트** | 게임당 최대 3회 |
| ❌ **오류 카운터** | 틀린 입력 횟수 표시 |
| 🔵 **셀 하이라이트** | 행·열·박스 및 같은 숫자 강조 |
| ⌨️ **키보드 지원** | 숫자 키, 방향키, Delete/Backspace |
| 📱 **모바일 최적화** | 터치 숫자패드 및 반응형 레이아웃 |
